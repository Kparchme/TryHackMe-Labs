# Windows Fundamentals 1

## Objective

Learn the basic features of the Windows operating system, including the desktop environment, file system, user accounts, and built-in administration tools.

## Topics Covered

- Windows Operating System
- Desktop Environment
- File Explorer
- User Accounts
- Control Panel
- Task Manager
- Windows Settings
- Windows File System

## Skills Demonstrated

- Navigated the Windows operating system
- Identified key Windows components
- Managed files and folders
- Used built-in Windows administration tools
- Explained core Windows concepts

## Key Takeaways

This lab provided a foundation in Windows administration and introduced core concepts used in IT support and cybersecurity.
