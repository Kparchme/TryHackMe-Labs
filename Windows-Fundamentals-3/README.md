# Windows Fundamentals 3

## Objective

Learn about Windows authentication, Microsoft Active Directory concepts, and the security mechanisms used to manage users, permissions, and domains in Windows environments.

## Topics Covered

- Authentication
- Active Directory Basics
- Domain Controllers
- Users and Groups
- Organizational Units (OUs)
- Group Policy
- Windows Permissions
- NTFS Permissions

## Skills Demonstrated

- Explained Windows authentication concepts
- Identified Active Directory components
- Understood the role of Domain Controllers
- Learned how users and groups are managed
- Explored Organizational Units (OUs)
- Examined Group Policy fundamentals
- Distinguished between share and NTFS permissions

## Key Takeaways

This lab introduced the fundamentals of Windows domain administration and Active Directory, providing a foundation for managing enterprise Windows environments.
