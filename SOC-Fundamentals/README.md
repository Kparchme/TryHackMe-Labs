# SOC Fundamentals

## Objective

Understand the role of a Security Operations Center (SOC), security monitoring, incident detection, and the tools analysts use to identify and respond to cybersecurity threats.

## Topics Covered

- Security Operations Center (SOC)
- Security Monitoring
- Alert Triage
- Incident Response
- Log Analysis
- Threat Detection
- Security Information and Event Management (SIEM)
- Blue Team Operations

## Skills Demonstrated

- Explained the purpose of a SOC
- Understood the responsibilities of SOC analysts
- Identified the stages of incident response
- Learned basic log analysis concepts
- Recognized common security alerts
- Explored threat detection techniques
- Understood the role of SIEM platforms

## Key Takeaways

This lab provided an introduction to Security Operations Center workflows and strengthened my understanding of how security teams detect, investigate, and respond to cyber threats.
