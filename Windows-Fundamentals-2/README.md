# Windows Fundamentals 2

## Objective

- System Configuration (MSConfig)
- User Account Control (UAC)

- Windows Registry
- Event Viewer

- Performance Monitor
- Windows Security

- Navigated Windows administrative tools
- Examined system configuration settings

- Identified key Windows security features
- Explained the purpose of the Windows Registry

This lab strengthened my understanding of Windows administration and security by introducing tools commonly used to monitor, configure, and troubleshoot Windows systems.
